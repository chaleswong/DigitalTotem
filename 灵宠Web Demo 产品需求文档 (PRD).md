## **一、 引言与项目背景 (Introduction & Project Background)**

### **1.1 时代演变与消费心理：全域中医的“新中式疗愈”大跃迁**

在宏观经济发展与现代都市高压生活的双重作用下，中国消费市场正迎来从“她经济”向“全员疗愈悦己经济”的深层次范式转移。

* **主力客群：** 26\~35 岁年轻人正成为新中式健康、茶饮与疗愈消费的中坚力量（占比已超 57%）。传统的“有病才治”被动就医模式，正在被“随时随地、主动防御、全面悦己”的高频生活化调理所取代。  
* **消费决策逆转：** 消费者不仅关注药效，更关注**产品颜值、使用氛围和松弛感**。手串、香囊等中医药衍生单品风靡，这种“腕上养生”不仅是物理层面的调理，更成为一种公开展示的新中式美学表达与社交“功德/松弛”标签。

### **1.2 理论渊源与学术支撑：上古“祝由术”与五感疗愈的现代科学转化**

全域中医心身疗愈方案的学理根源，可追溯至《黄帝内经》中记载的“祝由科”及“情志相胜”理论。

1. **祝由术的现代心身医学本质：** 《素问·移精变气论》指出：“古之治病，惟其移精变气，可祝由而已”。祝由的实质并不是迷信，而是通过安慰、共情、正向心理暗示，改变病人的整体精神状态，使气机运行恢复平衡。这与现代心理学中的“认知行为疗法（CBT）”完全契合。  
2. **五感调神的神经生物学通路：**  
   * **触觉与视觉通路（能量手串）：** 手串作为“文化锚定物”，在日常“盘玩”中，其材质微量元素和温润触觉能诱导大脑产生 $\\alpha$ 脑电波，消除烦躁；五行色彩的视觉输入，潜移默化地给予个体正向心理暗示。  
   * **嗅觉与听觉通路：** 通过天然芳香分子刺激大脑边缘系统调节神经递质；利用特定的声音频率（如五音疗法、颂钵）启动副交感神经，疏通气机。

## **二、 灵宠（电子宠物）核心构思与心理学锚点**

为了给冷冰冰的硬件赋予长效情感粘性，产品引入了“灵宠-气机精灵（五行化身）”这一核心功能：  
**“以形显气，心物共生。”**

* **心理学定位：** 满足人类的“投射效应（Projection）”与“养育本能（Nurturing Instinct）”**。灵宠不是一个需要用户喂食、铲屎的数字累赘，而是用户**自身心身状态的一面“数字镜子”。  
* **核心设定：** 灵宠的状态完全与用户的手串盘玩频次、情绪树洞吐槽、甚至所在地的环境天气绑定。用户气郁时它黯淡，用户静心时它发光。通过“拯救/养育灵宠”，用户在潜意识中完成了对自我内耗的救赎。

## **三、 行业典型标杆案例参考 (Benchmark Cases)**

在设计该灵宠的状态机与交互体验时，应深度参考并像素级借鉴以下互联网产品的成功模型：

### **1\. 多邻国（Duolingo）的标志性猫头鹰“多儿”（Duo）**

* **成功经验：** 将“状态机控制动态表情”做到极致。多儿会根据用户是否背单词、连续打卡天数，在首页呈现完全不同的形态（不打卡时它会哭泣、委屈、甚至发发带有催促感的小眼神；连续打卡时它变身金身、戴上墨镜、通体发光）。  
* **本产品借鉴点：** **“负反馈阻断机制”**。灵宠将完美复制多儿的“情绪触动”逻辑。当白领连续3天高压且不触碰手串时，灵宠呈现憔悴流散态，通过触发用户的“同理心与不忍抛弃感”，逼迫用户为了让宠物恢复能量而主动去盘珠、听白噪音，从而完成心理学上的行为纠偏。

### **2\. 潮汐 App (Tide) 的视觉流体**

* **成功经验：** 极其克制和留白的视觉设计。播放白噪音时，屏幕中央仅有一个配合呼吸做规律放大缩小的“半透明圆晕气流”。  
* **本产品借鉴点：** **去低幼化的新中式美学。** 灵宠彻底抛弃传统电子宠物（如拓麻歌子、精灵宝可梦）的具象动物形态，全面借鉴潮汐的“呼吸圆晕与极简流体”视觉，让灵宠成为高净值人群也乐于公开展示的高级视觉艺术。

## **四、 用户界面 (UI) 与视觉风格设计**

* **视觉形态：** 灵宠呈现为纯粹的、流线型的**半透明“气场光晕”**（如水墨流线或粒子风暴）或**温润古玉质感**的动态聚合体。  
* **界面降噪：** **取消任何传统的、大色块的对话框**。灵宠的言语直接以优雅的衬线字体（如宋体）半透明悬浮在光晕周围，或者通过优雅的底部抽屉（Bottom Sheet）轻量化输入。  
* **多模态引导：** 页面背景以慢速的图形光晕律动为主，自动加载高阶正念白噪音（432Hz 颂钵、古琴、空灵雨声），突显空灵、禅意与高级包裹感。

## **五、 业务方案 (Business Scheme)**

### **5.1 核心用户体验模块 (User Experience)**

1. **碰一碰 · 气机化形（激活与觉醒）：** 当手机碰一碰 NFC 手串时，系统读取设备 UID 唤起应用。AI 结合用户的初始体质测评，为其织造独一无二的五行属性灵宠（如：气郁质觉醒半透明墨色流线的“疏影墨灵”；心火旺觉醒内含流火高光的“炽焰玉灵”）。  
2. **抚摸盘玩 · 能量反哺（高频成瘾交互）：** 用户在物理世界盘动珠子，或在 Web 端直接点击/滑动主珠，前端画面中的灵宠光晕会立刻产生如同“被抚摸”般的水波纹荡开效果。灵宠通体泛起金光，使用户的物理行为在数字世界获得即时的、充满解压感的视觉与心理正向反馈。  
3. **大模型对话 · 灵宠解语（轻量情绪陪伴）：** 灵宠充当“情志树洞”的具象化载体。用户输入职场内耗或焦虑，灵宠光晕会产生向内收缩吸纳的动效（暗示吸走负能量），随后通过大模型输出符合其五行化身人设的、极具松弛感的金句进行化解。

### **5.2 用户粘性与长效留存机制 (Stickiness & Retention)**

1. **心身共筑的负反馈阻断机制（镜面投射）：** 灵宠的活跃状态直接投射用户的真实情绪。若系统检测到用户连续多日处于高压状态（如深夜连碰、吐槽高频）且不进行静心盘玩，灵宠的光晕会变得**黯淡、流线零散、甚至静止蜷缩**。  
2. **二十四节气形态演变（时令收集癖）：** 平台完全绑定传统二十四节气。每逢节气交替当天，灵宠会根据天地气机变换细节形态（如惊蛰时开出金色脉络，冬至时向内凝聚呈现温炉态），提供长期顺时陪伴的仪式感。

## **七、 大模型 AI Agent 层：基于 Coze 平台的设计方案**

### **7.1 长期记忆（Memory Bank \- Database 配置）**

在 Coze 平台中创建一个内置数据库 User\_Core\_Memory。包含字段：user\_id (String)、core\_topic (Text，自动提取并结构化用户面临的长期压力源，如“跨国合并项目/毕业论文”)、last\_body\_type (String，如“气郁质”)。

### **7.2 短期上下文（Variables 配置）**

全局变量配置：current\_time\_period（当前时段）、solar\_term（当前节气）、current\_stress\_score（当前压力分）。每次 NFC 碰一碰时通过 API 传参刷新。
